# Combine datasets and create derived features
print("=== Data Integration and Feature Engineering ===")

# Combine datasets
df_combined = pd.concat([df2019, df2020], ignore_index=True)
print(f"Combined dataset shape: {df_combined.shape}")

# Convert datetime columns
df_combined['start_time'] = pd.to_datetime(df_combined['start_time'])
df_combined['end_time'] = pd.to_datetime(df_combined['end_time'])

# Create derived features
df_combined['ride_length'] = (df_combined['end_time'] - df_combined['start_time']).dt.total_seconds() / 60.0
df_combined['day_of_week'] = df_combined['start_time'].dt.day_name()
df_combined['month'] = df_combined['start_time'].dt.month
df_combined['start_hour'] = df_combined['start_time'].dt.hour

# Standardize user type labels
df_combined["usertype"] = df_combined["usertype"].replace({
    "Subscriber": "member", 
    "Customer": "casual"
})

print("Feature engineering completed")
print("New columns:", ['ride_length', 'day_of_week', 'month', 'start_hour'])

print("=== Statistical Significance Test ===")

member_rides = df_combined[df_combined['usertype'] == 'member']['ride_length']

6. Strategic Recommendations¶

Based on the analysis of Cyclistic bike-share usage patterns, the following strategic recommendations are proposed to convert casual riders into annual members:

Marketing Strategy Recommendations¶

1. Targeted Weekend Promotions¶

  • Insight: Casual riders show 2.4x higher weekend usage compared to members
  • Action: Launch weekend-specific membership promotions highlighting convenience and cost savings
  • Implementation: "Weekend Warrior" membership tier with enhanced weekend benefits

2. Commuter Conversion Campaigns¶

  • Insight: Members predominantly use bikes for weekday commuting (shorter, frequent trips)
  • Action: Promote the economic benefits of membership for regular commuters
  • Implementation: "Commuter Challenge" - free trial memberships for frequent casual users

3. Duration-Based Value Proposition¶

  • Insight: Casual riders take 108% longer rides on average
  • Action: Emphasize unlimited ride time benefits of annual membership
  • Implementation: "Time Freedom" campaign showcasing cost savings for longer rides

4. Location-Specific Marketing¶

  • Insight: Different user types favor different station locations
  • Action: Deploy targeted advertising at high-casual-usage stations
  • Implementation: Digital displays and QR codes at top casual rider stations

Expected Business Impact¶

  • Revenue Growth: Converting 10% of casual riders could increase membership base by ~3.7%
  • Usage Optimization: More predictable demand patterns
  • Customer Lifetime Value: Higher retention and engagement rates

Next Steps¶

  1. A/B Testing: Pilot test marketing campaigns with control groups
  2. Customer Surveys: Gather direct feedback from casual riders about barriers to membership
  3. Seasonal Analysis: Extend analysis to full year data for seasonal patterns
  4. Pricing Strategy: Evaluate flexible pricing options (monthly, seasonal memberships)
